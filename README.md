# Moto BMW Canvas 2D

Aplicación web que dibuja una moto deportiva usando la API Canvas 2D de JavaScript.

## Características

- Uso de HTML, CSS, JS y Bootstrap
- Más de 30 figuras básicas
- Imagen original vs dibujo programado
- Código organizado por funciones
- Navbar y Footer
- Favicon personalizado

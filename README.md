🎯 Descripción del Proyecto

Este proyecto consiste en la construcción de una moto deportiva BMW geométrica utilizando la API Canvas 2D de HTML5, aplicando principios fundamentales de la graficación por computadora mediante figuras geométricas básicas.

La aplicación muestra:

🎨 Dibujo generado completamente por programación

🖼️ Imagen original de referencia

🧩 Uso de más de 30 figuras geométricas

🧠 Organización del código por funciones

💻 Diseño moderno con Bootstrap

📋 Requisitos del Examen

✔ Uso obligatorio de API Canvas 2D

✔ Uso de HTML, CSS y JavaScript

✔ Mínimo 30 figuras geométricas

✔ Interfaz web organizada y moderna

✔ Imagen original como referencia visual

🛠️ Tecnologías Utilizadas

HTML5

CSS3

JavaScript

Bootstrap 5

API Canvas 2D
